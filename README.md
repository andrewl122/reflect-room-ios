# Reflect Room

Reflect Room is an iOS self-reflection and journaling application designed to promote emotional awareness and personal growth through guided reflection.

## Features

- Mood tracking system
- Guided reflection prompts
- Secure local data storage using Core Data
- Dynamic gradient animations
- Gesture-based navigation for smooth user experience

## Tech Stack

- Swift
- SwiftUI
- Core Data
- AVKit

## Purpose

The goal of Reflect Room is to create a space where users can reflect on their daily experiences, track emotional patterns, and develop mindfulness through structured journaling.
