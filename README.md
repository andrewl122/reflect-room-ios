# Reflect Room

🚧 **Project Status: In Progress**

Reflect Room is an iOS self-reflection and journaling application designed to promote emotional awareness and personal growth through guided reflection and mood tracking. The app provides a calm digital space where users can document their thoughts, reflect on their day, and track emotional patterns over time.

## Features

- Daily journaling and reflection prompts
- Mood tracking system to monitor emotional patterns
- Secure local data storage using Core Data
- Dynamic gradient UI animations for a calming user experience
- Gesture-based navigation for smooth interaction

## Tech Stack

- Swift
- SwiftUI
- Core Data
- AVKit
- Xcode

## Purpose

The goal of Reflect Room is to create a simple and engaging tool that encourages users to reflect on their daily experiences and build mindfulness habits. The project focuses on combining thoughtful UI design with reliable local data storage to support a seamless journaling experience.
